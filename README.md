Employee Salary Insights
📌 Project Overview

Employee Salary Insights is a data analysis project developed using Python, Pandas, and Matplotlib. It analyzes employee salary data to generate meaningful insights about salary distribution, department performance, payroll costs, and workforce demographics.

🎯 Objectives

Analyze employee salary data.

Study salary distribution across employees.

Compare salaries by department and job role.

Analyze salary trends based on experience.

Calculate payroll costs by department.

Compare salaries by gender.

Visualize data using charts.

▶️ How to Run

Install the required libraries:

pip install pandas matplotlib

Place the dataset (employee_salary_dataset.csv) in the project folder.

Run the Python file:

python employee_salary_insights.py
